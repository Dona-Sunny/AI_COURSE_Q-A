# Introduction to Artificial Intelligence

Artificial Intelligence is the field of building systems that can perform tasks requiring human-like intelligence. These tasks include reasoning, learning, problem solving, and perception.

Machine learning is a subset of artificial intelligence in which systems learn patterns from data instead of being programmed with every rule explicitly.

Supervised learning uses labeled training data. The model learns a mapping from inputs to known outputs.

Neural networks are computational models inspired by the structure of the human brain. They consist of layers of interconnected nodes that transform input data into useful predictions.

Deep learning is a form of machine learning that uses many hidden layers in a neural network to learn complex patterns.
